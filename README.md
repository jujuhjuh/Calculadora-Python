# Calculadora-Python


* Projeto simples de uma calculadora em Python que funciona via terminal.
O usuário escolhe a operação desejada (soma, subtração, multiplicação ou divisão), informa dois valores e o programa exibe o resultado.

* O sistema roda em loop infinito até o usuário escolher a opção 0 – Sair.
Entradas inválidas são tratadas e o menu é exibido novamente após cada operação.
